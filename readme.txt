four methods implemeted in the code 

Program allows user to add a song creating a playlist, user can then 
delete song from the playlist, display the playlist and then play the 
playlist, with the option to press 1 to go the previous song and 2 to go the next song. 

was created using DoublyLinked List Data Structure. 

1. a method to add a song with its, song title, artist name, duration of song, id number and album it came if any. 

2. the second method is to be able to delete a song from the playlist 

3. the third method display the Playlist and shows all the songs with its corresponding information 

4. the fourth method plays the playlist and songs one song until 1 or 2 is entered, 1 allowing to the previous
song and 2 allowing to go the next song.
